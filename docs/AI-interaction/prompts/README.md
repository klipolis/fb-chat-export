# Prompt Examples for AI Interactions

This folder contains reusable prompt examples that help reproduce the project workflow and make AI interactions easier to reason about.

The prompts are written for this repository and focus on:

- reproducing the build and docs changes made in this project
- making requests that are clear, scoped, and actionable
- capturing AI interaction traces for education and future review

Files:

- `project-reproduction.md` — prompt examples for reproducing the project and code/docs changes.
- `trace-guidance.md` — guidance on recording AI interactions as trace documentation.
