# Terms and Conditions

This tool is designed to help you document your own Facebook Messenger conversations for personal summary purposes.

## Personal use only

- The exported text data should be used only for your own personal review and summary.
- Do not publish or distribute private conversation content without the consent of all participants.

## Facebook terms

Using this tool should not violate Facebook's Terms of Service:
https://www.facebook.com/legal/terms

## Privacy and ownership

The owner of the chat data is responsible for preserving privacy and protecting sensitive information.
This repository is not intended to collect, transmit, or publish other people's private conversation data.

## Recommended behavior

- Keep exported data local and secure.
- Use the tool only for personal archiving or summarization.
- Avoid using the tool to circumvent platform restrictions or share private messages.
